# 螺栓柔度计算器

这是一个基于 VDI 2230 柔度公式整理的本地网页版计算器。

## 使用方式

直接打开 `index.html` 即可使用，无需安装依赖或启动服务器。

## 当前功能

- 标准公制螺栓公称直径和螺距选择
- 自动计算中径 `d2` 和小径 `d3`
- 贯穿连接 / 盲孔连接切换
- 按 VDI 2230 截图公式计算拉伸柔度分项
- 输出拉伸柔度、弯曲柔度和轴向刚度
- 显示连接尺寸参考图和计算分段示意图

## 文件

- `index.html`: 网页计算器
- `assets/bolt-connection-diagram.png`: 螺栓连接尺寸示意图
- `螺栓柔度理论计算.pdf`: 原始参考资料
