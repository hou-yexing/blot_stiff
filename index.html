输入参数

光杆/颈缩分段

光杆/颈缩段按圆环截面计算拉伸面积：A = π(d² - dh²)/4。螺纹受载区长度自动取 max(lK - 光杆总长, 0)。

几何示意参数

当前 δSK 使用 AN = πd²/4，hh 与 dh 不参与头部拉伸柔度计算；它们用于尺寸图和空心/阶梯光杆核对。

公称直径和螺距来自常用 ISO 公制螺纹规格选项；选定后中径 d₂、小径 d₃ 按 d₂ = d - 0.64952P、d₃ = d - 1.22687P 自动更新。
